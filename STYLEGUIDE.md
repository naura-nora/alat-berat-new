# STYLE GUIDE

## 1. Color Palette
Aplikasi menggunakan warna berbasis Bootstrap untuk menjaga konsistensi dan kemudahan pengembangan.

- Primary / Action: Kuning (Bootstrap Warning)
- Danger: Merah (Bootstrap Danger)
- Info: Biru (Bootstrap Info)
- Sidebar: Abu-abu gelap (#343a40)
- Background: Putih
- Navbar: Putih
- Footer: Putih

## 2. Typography
- Menggunakan font default dari Bootstrap
- Heading menggunakan font tebal (bold)
- Isi teks menggunakan font normal (regular)

## 3. Button Style
Penggunaan tombol mengikuti standar Bootstrap:

- Tombol Login: btn-warning (kuning)
- Tombol Register: btn-warning (kuning)
- Tombol Tambah: btn-warning (kuning)
- Tombol Edit: btn-warning (kuning)
- Tombol Hapus: btn-danger (merah)
- Tombol Detail: btn-info (biru)

Semua tombol menggunakan bentuk standar Bootstrap dengan border radius default.

## 4. Form Input
- Menggunakan class `form-control` dari Bootstrap
- Validasi ditampilkan menggunakan pesan error (warna merah)
- Struktur form konsisten di setiap halaman

## 5. Layout
Struktur layout aplikasi:

- Navbar berada di bagian atas dengan warna putih
- Sidebar berada di sisi kiri dengan warna abu-abu gelap (#343a40)
- Content utama berada di tengah halaman
- Footer berada di bagian bawah dengan warna putih

## 6. UI Components
Komponen yang digunakan dalam aplikasi:

- Table: untuk menampilkan data (peminjaman, pengembalian, dll)
- Card: untuk tampilan dashboard
- Modal: untuk konfirmasi aksi (hapus / detail)
- Alert: untuk notifikasi sistem

## 7. Konsistensi Desain
Desain aplikasi dibuat konsisten dengan:
- Penggunaan warna yang sama untuk fungsi yang sama
- Layout yang seragam di setiap halaman
- Komponen Bootstrap digunakan secara konsisten

Warna kuning (warning) digunakan sebagai aksi utama agar mudah dikenali oleh pengguna, sedangkan warna merah digunakan untuk aksi berbahaya seperti penghapusan data.
